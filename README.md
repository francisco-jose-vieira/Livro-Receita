# 🍳 Minhas Receitas

Bem-vindo ao **Minhas Receitas**!  
Este projeto é um pequeno aplicativo desenvolvido em **Python** para gerenciar receitas de forma simples e prática.

Se você gosta de cozinhar (ou apenas de organizar suas receitas favoritas), este projeto pode ser um ótimo exemplo de aplicação feita com Python.

Abaixo está um guia rápido para rodar o projeto no seu computador. 🚀

---

## 📦 Pré-requisitos

Antes de começar, verifique se você tem as seguintes ferramentas instaladas:

- 🐍 **Python 3.13.5 ou superior**
- 💻 **Git** (para clonar o repositório)

---

## ⬇️ 1. Clonar o repositório

Primeiro, baixe o projeto para sua máquina.

Abra o terminal e execute:

```bash
git clone https://github.com/francisco-jose-vieira/Livro-Receita.git
```

Isso criará uma cópia do projeto no seu computador.

---

## 📂 2. Acessar a pasta do projeto

Entre na pasta que foi criada:

```bash
cd Livro-Receita
```

---

## 📚 3. Instalar as dependências

O projeto utiliza algumas bibliotecas externas.  
Para instalá-las, execute:

```bash
pip install customtkinter pillow
```

Essas bibliotecas são responsáveis pela **interface gráfica** e **manipulação de imagens**.

---

## ▶️ 4. Executar o projeto

Agora vem a melhor parte: **rodar o programa!**

Execute o seguinte comando:

```bash
python main.py
```

Se tudo estiver correto, o aplicativo abrirá e você poderá começar a explorar as receitas. 🍲

---

## 👨‍💻 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- 🐍 **Python**
- 🖼 **CustomTkinter**
- 🖼 **Pillow**
